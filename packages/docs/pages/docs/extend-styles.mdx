# Extend and share styles

Since your CSS are objects it's easy and intuitive to extend and share styles using the spread operator.

Extend a Panache component.

```jsx
const MyComponent = panache.div({
	color: 'blue',
	backgroundColor: 'gray',
})

const MyOtherComponent = panache.div({
	...MyComponent.Styles,
	color: 'red'
})
```

You can also extend components styles with your own style objects, or objects from your theme.

```jsx
const errorStyles = {
	backgroundColor: 'red',
	color: '#fff',
}

const MyThing = panache.div(({ hasError }) => ({
	... hasError && errorStyles,
	padding: '15px'
}))
```
